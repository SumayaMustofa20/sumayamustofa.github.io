Project Update Guidelines

Step 1

Edit HTML File

Open projects.html in VS Code and locate the project card you want to update. Modify the content, status, or add new project cards following the existing structure.

Step 2

Update Project Status

Change the project-status class and text: "Completed", "In Progress", or "Planned". Update project links from "disabled" to actual links when ready.

Step 3

Deploy Changes

Save your changes, then run: git add ., git commit -m "Update projects", git push origin main. Your changes will be live in 1-2 minutes.
