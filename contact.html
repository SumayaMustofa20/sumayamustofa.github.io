Contact Form Setup Guide

Option 1

Formspree (Recommended)

Sign up at formspree.io, verify your email, and replace the form action with your Formspree endpoint: action="https://formspree.io/f/YOUR_FORM_ID"

Option 2

EmailJS

Use EmailJS for client-side email sending. Add EmailJS script and configure the form submission handler in the JavaScript file.

Option 3

Netlify Forms

If you deploy on Netlify instead of GitHub Pages, add netlify attribute to form tag and Netlify will handle form submissions automatically.
